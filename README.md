# 🏨 酒店美誉度管理SUITE | Hotel Reputation Management Suite

> 通用版 — 适用于任何酒店集团/品牌/单体酒店
>
> **触发词**：美誉度管理 / 差评分析 / 舆情监控 / 酒店口碑 / 整改工单

## ✨ 简介

酒店差评采集 + AI分类 + 根因分析 + 整改闭环 + 周报统计

无论您管理一家单体酒店还是整个酒店集团，只需配置酒店参数，即可快速上线差评管理能力。

## 🚀 快速开始

### 1. 填写配置区

编辑 `SKILL.md` 顶部的配置区，填写您的酒店信息：

```yaml
hotel_config:
  hotel_name: "XX酒店"        # 酒店全称
  hotel_code: "BJ"            # 缩写（工单编号用）
  city: "北京"                # 城市
  brand: "XX酒店集团"          # 品牌
  
managers:
  - name: "张三"
    open_id: "ou_xxxxxxxx"    # 飞书open_id
```

### 2. 触发差评分析

```
对AI说：「分析XX酒店差评」
```

AI自动执行：采集 → 分类 → 生成工单 → 推送负责人

## 📊 示例输出（去标识化）

```
📊 差评分析报告 | XX酒店 | 2026年4月19日

数据来源：携程 / Trip.com / 去哪儿 / 小红书
采集差评：26条

---
🔴 重点差评工单（严重度≥7）

工单编号：DR-20260419-BJ-001
酒店名称：XX酒店
平台：Trip.com
发布时间：2026年4月
入住日期：2026年4月
差评分：1/5
严重度：9/10
是否已回复：❌ 否

原文摘要：凌晨客房断电、要求加拖鞋20分钟未送、前台态度不耐烦

---
5Why根因分析：
WHY1：前台人手不足 → 凌晨只有1人值班
WHY2：工作流程缺乏优先级 → 没有区分紧急/普通需求
WHY3：服务补救意识不足 → 员工未主动致歉
WHY4：激励机制缺失 → 差评不影响绩效考核
WHY5：缺乏夜间服务SOP → 没有明确的响应时效要求

---
整改要求：
1. 【立即】制定夜间客房服务SOP（10分钟内响应）
2. 【短期】将差评纳入前台KPI考核
3. 【长期】管理组每日抽检夜间服务录音
```
