# 酒店美誉度管理SUITE

> 酒店差评采集 + AI分类 + 根因分析 + 整改闭环 + 周报统计

**适用场景**：酒店集团美誉度管理、OTA差评跟进、舆情整改闭环

**触发词**：美誉度管理 / 差评分析 / 舆情监控 / 酒店口碑 / 整改工单 / 差评跟进

---

## 核心功能

- **自动采集**：携程 / Trip.com / 去哪儿 / 小红书 / Klook / Expedia 全渠道
- **智能分类**：12维度AI分类 + 严重度1-10分
- **根因追溯**：每条差评配5Why深度分析
- **整改闭环**：工单分配 → 24h回复 → 7日整改 → 闭环存档
- **一键复用**：换酒店只需改配置区，5分钟上线

---

## 快速开始

### 1. 填写配置区

编辑 `SKILL.md` 顶部的配置区，填写酒店信息：

```yaml
hotel_config:
  hotel_name: "XX酒店"        # 酒店全称
  hotel_code: "BJ"            # 缩写（工单编号用）
  city: "北京"                # 城市
  brand: "XX酒店集团"          # 品牌
  
managers:
  - name: "张三"
    open_id: "ou_xxxxxxxx"    # 飞书open_id
```

### 2. 触发差评分析

对AI说：`分析XX酒店差评`

AI自动执行：采集 → 分类 → 生成工单 → 推送负责人
